# Gemini 활용 기록

> 이번 수행평가에서는 학교 계정 Gemini 사용을 허용합니다.  
> 단, Gemini가 만든 코드를 이해하지 못한 채 그대로 제출하면 안 됩니다.  
> 실제 개인정보, 친구 이름, 성적, 전화번호, 주소 등은 절대 입력하지 않습니다.

## 1차 Gemini 활용

- 사용 날짜:2026-06-16
- 사용 목적:슬롯머신 게임 프로그램 구현 및 함수 구성 도움 받기
- 내가 입력한 프롬프트:

```text
2차원 리스트를 활용한 파이썬 슬롯머신 게임을 만들어줘.
함수를 사용해서 결과 출력, 보상 계산, 슬롯 회전 기능을 나눠서 작성해줘.
```

- Gemini가 제안한 내용 요약:2차원 리스트에 슬롯 그림을 저장하는 방법 제안
슬롯 회전 함수(spin_slot) 구성 방법 제안
보상 계산 함수(calculate_reward) 작성 방법 제안
결과 출력 함수(print_result) 분리 제안
- 내가 반영한 부분:22차원 리스트 사용
함수별 역할 분리
랜덤으로 슬롯 결과 생성
2개 일치, 3개 일치 보상 지급
- 내가 수정하거나 사용하지 않은 부분:출력 문구를 직접 수정함
보상 배율을 게임에 맞게 조정함
입력 및 종료 조건을 추가함
- 반영 위치: spin_slot(), calculate_reward(), print_result() 함수

## 2차 Gemini 활용

- 사용 날짜:2026-06-16
- 사용 목적:입력 오류 처리 개선
- 내가 입력한 프롬프트:

```text
슬롯머신 게임에서 숫자가 아닌 값을 입력해도 오류가 나지 않게 수정해줘.
```

- Gemini가 제안한 내용 요약:try-except 문을 사용한 예외 처리 방법 제안
- 내가 반영한 부분:숫자가 아닌 값을 입력하면 다시 입력받도록 수정
- 내가 수정하거나 사용하지 않은 부분:출력 문구
- 반영 위치:main() 함수의 베팅 금액 입력 부분

## AI 활용 성찰

Gemini가 도움이 된 점:함수 분리 방법과 2차원 리스트 활용 방법을 이해하는 데 도움이 되었다. 또한 입력 오류 처리와 코드 정리 방법을 배울 수 있었다.


Gemini의 제안을 그대로 쓰지 않고 내가 판단하거나 수정한 점:출력 문구와 게임 규칙을 직접 수정하였고, 종료 조건과 일부 예외 처리 방법을 추가하였다. 제안된 내용을 그대로 사용하지 않고 프로그램에 맞게 변경하였다.


다음에 AI를 사용할 때 주의해야 할 점:AI가 제안한 내용을 그대로 사용하지 않고 직접 이해한 후 수정해야 한다. 또한 코드가 제대로 동작하는지 직접 실행하여 확인해야 한다.
