# 프로젝트 계획서

## 1. 프로젝트 주제

내가 만들 프로그램의 주제:슬롯머신 게임 프로그램

예) 기분과 시간에 따른 활동 추천기

---

## 2. 문제 상황

이 프로그램이 해결하려는 문제를 설명하세요.

예) 사용자가 현재 기분과 사용할 수 있는 시간을 입력하면, 조건에 맞는 활동을 추천해 주고 싶다.

내 설명:사용자가 가상 돈을 가지고 슬롯머신 게임을 즐길 수 있도록 한다. 슬롯을 돌려 나온 그림의 조합에 따라 당첨 여부와 획득 금액을 계산하여 게임의 재미를 제공하고 싶다.


---

## 3. 입력-처리-출력 구조

### 입력

사용자가 입력하거나 프로그램이 사용할 데이터는 무엇인가?

- 게임 시작 여부
- 슬롯 1회 이용 금액
- 슬롯 그림(과일, 별, 보석 등) 데이터

### 처리

입력된 데이터를 어떤 규칙이나 알고리즘으로 처리할 것인가?

- 랜덤으로 슬롯 그림 3개를 선택한다.
- 같은 그림이 몇 개 나왔는지 확인한다.
- 당첨 규칙에 따라 획득 금액을 계산하고 소지금을 갱신한다.
### 출력

프로그램이 최종적으로 무엇을 출력하는가?

- 슬롯 결과 그림 3개
- 당첨 여부와 현재 소지금

## 6. 사용할 제어구조

### 조건문을 사용할 부분

예) 사용자의 예산보다 가격이 낮은 항목만 추천한다.

내 계획:슬롯 결과가 모두 같으면 큰 상금을 지급하고, 2개가 같으면 작은 상금을 지급하며, 모두 다르면 꽝으로 처리한다.


### 반복문을 사용할 부분

예) 2차원 리스트의 모든 항목을 하나씩 검사한다.

내 계획:슬롯 그림 3개를 랜덤으로 뽑을 때 반복문을 사용하고, 게임을 여러 번 진행할 수 있도록 반복문을 사용한다.
